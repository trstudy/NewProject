Files required to build hexamer tabel:
 * Human_coding_transcripts_CDS.fa
 * Human_noncoding_transcripts_RNA.fa


Files required to build logit model:
 * Human_coding_transcripts_mRNA.fa
 * Human_noncoding_transcripts_RNA.fa 
 or
 * Human_coding_transcripts_hg19.bed
 * Human_noncoding_transcripts_hg19.bed
 NOTE: reference genome sequence is required if input bed files
 

Files  required to test CPAT
 * Human_test_coding_mRNA.fa
 * Human_test_noncoding_RNA.fa
 
 or 
 
 * Human_test_coding_mRNA_hg19.bed
 * Human_test_noncoding_hg19.bed
 NOTE: reference genome sequence is required if input bed files
 
Read online manual for detailed instructions:
